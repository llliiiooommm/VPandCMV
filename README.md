# Lab 7 — Книжные карточки на React

## Что за зверь

React-приложение, которое показывает карточки книг. В каждой карточке:
- Обложка (приходит как BLOB, чисто по заданию)
- Название книги (жирный, крупный шрифт)
- Авторы (поменьше, серенькие)

Карточки располагаются горизонтально и переносятся на следующую строку, если не влезают. Никаких таблиц — только flexbox.

## Откуда данные

Изначально в задании был `fakeapi.extendsclass.com`, но он отдавал 403 (доступ запрещён). Поэтому я пошёл другим путём:

- **Open Library API** — отдаёт список книг и ID обложек
- **Open Library Cover API** — тянет обложки по ID в виде BLOB
- **Google Books API** — запасной вариант, если у книги нет обложки в Open Library

Обложка грузится как BLOB → превращается в URL через `URL.createObjectURL()` → показывается в `<img>`.

## Как запустить

```bash
npm install
npm run dev
```
Откроется `http://localhost:NNNN`

## Что умеет

- 🔍 Поиск книг по названию (мгновенный, без кнопок)
