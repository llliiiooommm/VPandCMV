## Описание
Реализация типобезопасного конвейера преобразований над массивами объектов на TypeScript.

### Основные возможности

- **Типы:** `Transform`, `Where`, `Sort`, `Group`, `GroupBy`, `GroupTransform`, `Having`
- **Функция `query`** — создаёт конвейер из нескольких шагов
- **Фабрики:** `where()`, `sort()`, `groupBy()`, `having()`
